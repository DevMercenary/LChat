# Установка LChat

LChat нужно поставить на **оба** компьютера (Fedora и Windows 11). Ниже — по шагам
для каждой ОС. Единственное общее требование — установленный **Rust** (`rustup`,
https://rust-lang.org).

---

## Fedora / Linux

```bash
cd LChat
./install.sh
```

Скрипт:
- соберёт release-бинарник (`cargo build --release`);
- поставит его в `~/.local/bin/lchat`;
- добавит ярлык «LChat» в меню приложений;
- подскажет команды для брандмауэра и недостающие пакеты.

Запуск: команда `lchat` в терминале или пункт **«LChat»** в меню приложений.

### Необязательные пакеты

- **Вставка скопированных файлов** из файлового менеджера (Ctrl+V / «📋 Из буфера»):
  - Wayland: `sudo dnf install wl-clipboard`
  - X11: `sudo dnf install xclip`
  - *(картинки из буфера работают и без них)*
- **Миниатюры видео:** `sudo dnf install ffmpeg`

### Брандмауэр (один раз)

```bash
sudo firewall-cmd --add-port=9009/tcp --permanent
sudo firewall-cmd --add-port=9010/udp --permanent
sudo firewall-cmd --reload
```

## Windows 11

### Просто скачать (без Rust) — рекомендуется

Зайди в [Releases](https://github.com/DevMercenary/LChat/releases), скачай
`LChat-windows-x86_64.zip`, распакуй и запусти `lchat.exe`. Rust ставить не нужно.
Для ярлыка в «Пуск» запусти `install.ps1` из архива (правой кнопкой → «Выполнить
с помощью PowerShell»).

### Собрать из исходников

1. Установи Rust: https://rust-lang.org → «rustup».
2. Скопируй папку `LChat` на компьютер.
3. Правой кнопкой по `install.ps1` → **«Выполнить с помощью PowerShell»**.
   Либо в PowerShell:
   ```powershell
   powershell -ExecutionPolicy Bypass -File install.ps1
   ```

Скрипт соберёт `lchat.exe`, положит его в
`%LOCALAPPDATA%\Programs\LChat\` и создаст ярлык **«LChat»** в меню «Пуск».

- Если запустить PowerShell **от имени администратора** — скрипт сразу откроет
  порты 9009/TCP и 9010/UDP в брандмауэре.
- Если без прав администратора — при первом запуске LChat Защитник Windows
  спросит доступ к сети: разреши для **частных сетей**.
- Для миниатюр видео поставь `ffmpeg` и добавь его в `PATH`.

## Автозапуск при старте системы

Не требует переустановки: включается **галочкой «Запускать при старте системы»**
прямо в окне LChat (для текущего пользователя, без прав администратора).

- Linux: создаётся `~/.config/autostart/lchat.desktop`.
- Windows: ключ в реестре `HKCU\...\CurrentVersion\Run`.

> Автозапуск ссылается на текущее расположение бинарника. Если переустановишь в
> другое место — выключи и включи галочку заново.

---

## Первый запуск

1. Запусти LChat на обоих компах.
2. Задай **одинаковый пароль** в поле «Пароль».
3. Подключись: кнопкой соседа в «Найдены в сети» (автопоиск) или впиши IP вручную.

Настройки и история чата сохраняются в:
- Linux: `~/.config/lchat/config.json`
- Windows: `%APPDATA%\lchat\config.json`

Историю можно отключить галочкой «Сохранять историю чата» и стереть кнопкой
«🧹 Очистить чат».
